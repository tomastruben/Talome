# Completion Checklist

Before finishing, verify:

- the app spec is valid and complete
- compose services, ports, volumes, and env vars are internally consistent
- scaffold files exist when scaffold generation is enabled
- the output reflects Talome's design language
- reused sources are recorded in metadata
- the result is specific to the user request, not a generic boilerplate
- the generated app is realistic enough to run or to be corrected with only small follow-up tweaks
