# Launch Materials

Draft copy, checklists, and assets for the Talome public launch.

| File | Purpose |
|---|---|
| [`../launch-plan.md`](../launch-plan.md) | Full launch plan, gaps audit, T-minus timeline, risk register |
| [`show-hn.md`](./show-hn.md) | Show HN title and first comment |
| [`reddit.md`](./reddit.md) | r/selfhosted, r/homelab, r/unraid post drafts |
| [`twitter-thread.md`](./twitter-thread.md) | 9-tweet X launch thread |
| [`product-hunt.md`](./product-hunt.md) | Product Hunt tagline, description, maker comment, gallery plan |

## How to use

1. Read [`../launch-plan.md`](../launch-plan.md) first — it's the master checklist.
2. Fill in the blanks in each draft (stars baseline, account handles, demo video URL).
3. Run through the T-14, T-7, T-3, T-1 checklists in the plan.
4. On launch day, paste the drafts into their destinations and be online to reply.

None of these are final. Iterate on the copy until it feels true to the product, not a marketing voice.
